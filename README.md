Творческое задание
Вам поставлена задача разработки системы для генерации ответов на обращения. 
В качестве решения предлагается представить предложения по реализации, в котором рекомендуется проработать следующие вопросы: 
	стек используемых технологий
	поэтапный плана реализации 
	верхне-уровневое описание архитектуры.
Генерируемый ответ должен одновременно соответствовать следующим требованиям:
1.	Все вопросы / аргументы заявителя должны найти отражение в ответе.
2.	При подготовке ответа стоит учитывать позицию организации, на которую жалуется заявитель.
3.	Ответ не должен противоречить требованиям НПА и содержать ссылки на них 
4.	Ответ должен соответствовать требованиям по делопроизводству.
Обращения поступают в электронном виде, а также в форме сканов.

В качестве альтернативы или дополнения можно аналогично предложить решение задач входной классификации обращений по тематикам и выделение именованных сущностей (ФИО, наименование организации, даты и адреса).
